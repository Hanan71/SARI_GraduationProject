## Packages
recharts | For real-time data visualization and charts
framer-motion | For smooth page transitions and UI animations
lucide-react | Icon set for the industrial dashboard interface
date-fns | For formatting timestamps in charts and logs

## Notes
- The app uses a dark mode industrial aesthetic by default
- Data fetching uses refetchInterval: 1000 to simulate real-time sensor updates
- Authentication is handled via Replit Auth (use-auth.ts)
- Simulation controls interact with the backend to change data generation patterns
